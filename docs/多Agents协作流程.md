


### 协作流程 示例：用户问"写一篇关于人工智能的文章"


第 1 轮：PlannerAgent 规划
┌─────────────────────────────────────────┐
│ Reasoning（推理）                  │
│ - 分析任务类型：长文本写作      │
│ - 需要收集信息              │
│ - 需要调用 writer 生成文章    │
└─────────────────────────────────────────┘
         │
         ▼
    Acting（行动）
┌─────────────────────────────────────────┐
│ 调用工具：                        │
│ assign_multi_subjective_tasks_to_info_seeker│
│ 参数：tasks = [                 │
│   {"content": "搜索AI定义"},   │
│   {"content": "搜索AI应用"}      │
│ ]                                 │
└─────────────────────────────────────────┘

第 2 轮：InfoSeeker 并行执行
┌────────────────┬────────────────┐
│ Task 1      │  Task 2       │
│ 搜索AI定义   │  搜索AI应用    │
└────────────────┴────────────────┘
         │             │
         ▼             ▼
    返回结果1    返回结果2

第 3 轮：PlannerAgent 收集结果并规划
┌─────────────────────────────────────────┐
│ Reasoning（推理）                  │
│ - 信息收集完成                │
│ - 现在可以调用 writer 了      │
└─────────────────────────────────────────┘
         │
         ▼
    Acting（行动）
┌─────────────────────────────────────────┐
│ 调用工具：                        │
│ assign_subjective_task_to_writer      │
│ 参数：                            │
│ task_content = "基于收集的信息   │
│ 写一篇关于人工智能的文章"          │
│ key_files = [                     │
│   {                                │
│     "file_path": "./data/info1.json",│
│     "desc": "AI定义信息"        │
│   }, {                            │
│     "file_path": "./data/info2.json",│
│     "desc": "AI应用信息"        │
│   }                               │
│ ]                                 │
└─────────────────────────────────────────┘

第 4 轮：WriterAgent 生成内容
┌─────────────────────────────────────────┐
│ Reasoning（推理）                  │
│ - 分析收集的信息              │
│ - 规划文章结构                │
│ - 逐步生成内容                  │
└─────────────────────────────────────────┘
         │
         ▼
    Acting（行动）
┌─────────────────────────────────────────┐
│ 调用工具：                        │
│ file_write                        │
│ 写入文章草稿                      │
│ section_writer                    │
│ 写入各个章节                      │
└─────────────────────────────────────────┘
         │
         ▼
    返回最终文章

第 5 轮：PlannerAgent 完成任务
┌─────────────────────────────────────────┐
│ Reasoning（推理）                  │
│ - 确认任务完成                │
└─────────────────────────────────────────┘
         │
         ▼
    Acting（行动）
┌─────────────────────────────────────────┐
│ 调用工具：                        │
│ planner_subjective_task_done        │
│ 参数：                            │
│ final_answer = "文章已生成完成"  │
└─────────────────────────────────────────┘
